## Descripción

Software desarrollado en Python para gestionar préstamos de artículos utilizando archivos planos.

El sistema permite:

- Registrar usuarios
- Registrar artículos
- Gestionar préstamos
- Registrar devoluciones
- Generar ventas automáticas
- Exportar información a CSV
- Administrar inventario
